## 📏Como baixar o projeto
```bash

   # clonar repositório
   $ git clone https://github.com/EnzoVizotto/alura

   # entrar no diretório
   $ cd alura

   # instalar as dependencias
   $ yarn install

   #iniciar o projeto
   $ yarn start

```
